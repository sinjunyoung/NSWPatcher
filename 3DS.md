3DS Seedminer로 movable.sed 구하는 방법

1. 준비물

- 3DS SD카드를 컴퓨터에 연결하고 Nintendo 3DS 폴더 안에 있는 32자리 난수 폴더 이름(ID0)을 복사함.
- 본인 3DS 기기의 친구 코드를 확인함. (하이픈 제외, 순수 숫자만 복사)

2. 사이트 접속

- Bruteforcemovable 사이트( https://bruteforcemovable.com/ )나 Seedminer 사이트( https://seedminer.hacks.guide/ )에 접속함.

3. 앞서 준비한 친구 코드와 ID0를 각각 입력란에 넣고 GO 버튼을 누름.

4. 봇 친구 추가

- GO를 누르면 화면에 사이트가 지정해 준 봇 친구 코드가 생성됨.
- 3DS 기기를 켜고 인터넷에 연결한 뒤, 해당 봇 코드를 내 기기에 친구 등록함.

5. 연산 및 다운로드

- 친구 추가가 정상적으로 완료되면 사이트가 알아서 인식하고 자동으로 연산을 시작함.
- 연산이 끝나면 Download movable.sed 버튼이 활성화되며, 해당 파일을 다운로드하면 완료됨.

  참고: 이전에 이미 진행했던 기기라면 중간 과정 다 건너뛰고 바로 Done 상태로 넘어감. 이 경우 별도 작업 없이 바로 다운로드 버튼을 눌러 파일을 받으면 됨.
